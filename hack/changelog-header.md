### Linux

```shell
curl -L https://github.com/jenkins-x-plugins/jx-admin/releases/download/v{{.Version}}/jx-admin-linux-amd64.tar.gz | tar xzv 
sudo mv jx-admin /usr/local/bin
```

### macOS

```shell
curl -L  https://github.com/jenkins-x-plugins/jx-admin/releases/download/v{{.Version}}/jx-admin-darwin-amd64.tar.gz | tar xzv
sudo mv jx-admin /usr/local/bin
```
